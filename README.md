# 🏦 Banking System – Console Application

## 📌 Overview
A console-based **Banking System** built in **C++** using **Object-Oriented Programming (OOP)** and **file handling**.

This system simulates real-world banking operations with a strong focus on **security, permissions, validated transactions, and data integrity**.

The project allows managing **users, clients, accounts, deposits, withdrawals, bank transfers, and currency exchange**, while tracking and auditing all actions.

---

# 🚀 Features

## 👤 User Management System
- Admin & normal users  
- **Role-Based Access Control (RBAC)**  
- Admins have full control  
- Normal users have restricted permissions  

---

## 👥 Client Management
- ➕ Add a new client  
- ✏️ Update client information  
- ❌ Delete an existing client  
- 📋 Show all client details  

---

## 💳 Account & Transactions
- Bank accounts per client with balance tracking  
- 💰 Deposit money into an account  
- 💸 Withdraw money from an account  
- 🔁 Bank transfers with full validation & atomic execution  

---

## ✅ Transfer Logging
Every transfer is logged with:

- Sender account (**From**)  
- Receiver account (**To**)  
- Transfer amount  
- User performing the transfer  
- Date & time  

---

## 💱 Currency Exchange System (NEW)
A new module that simulates a **Currency Exchange System** integrated with the banking system.

### Features
- 💹 **Currency Listing Screen**  
  Displays all available currencies and their exchange rates.

- 🔍 **Find Currency Screen**  
  Search for currencies by **code or country name**.

- ✏️ **Update Currency Rate Screen**  
  Update exchange rates dynamically.

- 🔄 **Currency Calculator Screen**  
  Convert between any two currencies instantly using **USD as a base currency**.

This module demonstrates **reusability, modular design, and clean separation of responsibilities using OOP principles.**

## 🧾 Activity Logging & Auditing
